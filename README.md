<h2>About app</h2>
<p>Real time chat is a basic chat app where you communicate with other people. It features authorization system, changing username
and profile picture, sending messages to a global chat.</p>
<h2>Technologies</h2>
<ul>
<li>React</li>
<li>TypeScript</li>
<li>Firebase</li>
</ul>
